# Tic-Tac-Toe

📌 Overview
This is a simple Tic Tac Toe game implemented in C++.

The player uses X and the computer uses O.

The game runs in the console and allows the player to compete against a computer making random moves.

It demonstrates arrays, functions, loops, conditionals, and basic AI logic in C++.

⚙️ Features
Draws a 3x3 Tic Tac Toe board in the console.

Player chooses a position (1–9) to place their marker.

Computer randomly selects an available position.

Checks for winner (rows, columns, diagonals).

Detects tie when all spaces are filled.

Displays the result: YOU WIN!, YOU LOSE!, or IT'S A TIE!
